# OpenHouse

OpenHouse gives Tytus pods and third-party AI agents a house, body, room, mood, and live operational status inside Tytus OS.

Architecture: standalone Tytus installed app loaded through `tytus-app.json` `entry.url`.

Runtime: React + SVG, Host API, per-app SQLite.

No Phaser, no Three.js, no Star-Office sidecar, no persisted raw secrets.
